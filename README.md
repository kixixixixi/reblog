# Reblog
Generate json for static frontend blog from markdown.

# Getting Started
## Installation
```
$ npm i reblog --save
```
Or install global
```
$ npm i -g reblog
```

## Init
Create `articles` directory and config file.
```
$ reblog init 
```

## Create aritcle
```
$ reblog new <title>
```

## Generate json
```
$ reblog build
```
