# Reblog
Generate json for static frontend blog from markdown.

# Getting Started
## Installation
```
$ npm i reblog --save
```
Or install global
```
$ npm i -g reblog
```

## Init
Create `articles` directory and config file.
```
$ npx reblog init 
```

## Deploy blog aritcle (`*.md`)
Deploy your blog articles to `./articles` directory.

## Generate json
```
$ npx reblog build
```
